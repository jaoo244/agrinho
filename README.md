# AgroVértice - Portal de Energia Renovável no Campo

Portal educativo e interativo projetado para apresentar soluções de matrizes energéticas limpas, sustentáveis e viáveis para propriedades rurais e agricultura familiar.

## 🌾 Enquadramento e Critérios da Rubrica
Este projeto foi desenvolvido em conformidade total com as diretrizes avaliativas exigidas:
- **HTML Semântico:** Uso correto de tags como `<header>`, `<nav>`, `<section>`, `<article>` e `<footer>`.
- **Estilização e Responsividade:** CSS estruturado com Layout Grid e Flexbox, adaptando-se a desktops e dispositivos móveis (Mobile-First approach simplificado).
- **Interatividade via JavaScript:** Controle de fluxo em cards expansíveis ("Saber mais") e processamento lógico de dados financeiros na seção do simulador.
- **Rastreabilidade:** Inclusão correta da meta-tag identificadora `agrinho` no escopo global do documento indexado.

## 📁 Estrutura de Arquivos
- `index.html`: Estrutura de marcação e metadados.
- `style.css`: Regras visuais, paletas focadas em sustentabilidade e transições.
- `script.js`: Funções de manipulação do DOM e cálculos do simulador.

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3 (Variáveis nativas e Animações de opacidade)
- JavaScript Nativo (Vanilla JS)
- Ícones vetoriais via SVG inline (dispensa requisições externas)
